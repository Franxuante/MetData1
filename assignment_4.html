Question 2: Calculating Moran’s I

Write a function calculate_morans_i that:

Takes a spatial weight matrix W and an attribute array y.
Calculates Moran’s I for the given attribute array.
Returns the Moran’s I value and its p-value.
#Defining the function
def calculate_morans_i():
    w_db = weights.Queen.from_dataframe(db, ids='CBS_Buurtcode') #Standardized weight is not needed since it won't be presented visually
    y = db['Oppervlakte_m2'] #Attributed array 
    mi = esda.Moran(y , w_db) #Function to calculate Morans I
    mi_I = mi.I
    mi_p = mi.p_sim #Function for calculating the P-value
    return mi_I, mi_p #Returns Morans I and P-value
