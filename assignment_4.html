Question 2: Calculating Moran’s I¶

Write a function calculate_morans_i that:

Takes a spatial weight matrix W and an attribute array y.
Calculates Moran’s I for the given attribute array.
Returns the Moran’s I value and its p-value.
In [9]:
#Defining the function
def calculate_morans_i():
    w_db = weights.Queen.from_dataframe(db, ids='CBS_Buurtcode') #Standardized weight is not needed since it won't be presented visually, the index is so it can be called upon later.
    y = db['Oppervlakte_m2'] #Attributed array 
    mi = esda.Moran(y , w_db) #Function to calculate Morans I
    mi_I = mi.I
    mi_p = mi.p_sim #Function for calculating the P-value
    return mi_I, mi_p #Returns Morans I and P-value

#Output check

db['centr'] = db.geometry.centroid 

#Create a copy of db centr_db for subsequent operations.
centr_db = db.copy() 

#Set the geometry to the centroids defined in the new column
centr_f = centr_db.set_geometry('centr', inplace=True)  

#Defining function
def create_distance_weights():
    w_dist = weights.DistanceBand.from_dataframe(centr_f, threshold=500, ids='CBS_Buurtcode', binary=False) #The function calls upon creating a circle(threshold) around a certain point. Alle the centroids that fall within get a weight based upon how far they are from the center polygon (binary=False)
    return w_dist
